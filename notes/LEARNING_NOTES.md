# Learning Notes

## RAG

RAG means Retrieval-Augmented Generation. The app retrieves relevant data first, then uses an LLM to generate a grounded answer.

## Multi-source RAG

Multi-source RAG retrieves from different places, such as PDFs, SQL databases, APIs, and logs.

## Agentic RAG

Agentic RAG means the AI can decide which tools to call before answering. In this project, the agent may call tools for contract search, schedule lookup, site notes, weather records, and change-order history.

## MVP

MVP means Minimum Viable Product. It is the smallest working version that proves the main idea.
