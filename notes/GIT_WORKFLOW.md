# Git Workflow

## First commit

```bash
git init
git add .
git commit -m "Initial Alter Built AI starter project"
```

## Connect to GitHub

```bash
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/alter-built-ai.git
git push -u origin main
```

## Daily workflow

```bash
git status
git add .
git commit -m "Describe the completed step"
git push
```
